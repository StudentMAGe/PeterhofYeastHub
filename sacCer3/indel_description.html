Description

These tracks contain information about short indels variations in several Saccharomyces cerevisiae strains of the Peterhof genetic collection. For more information on strains please see the description page.

Methods

Short reads were aligned with bowtie v2.1.0; SNV calling was performed with samtools v1.0 mpileup. Only indels with quality above 30 and supported with all reads are shown. Files were annotated with SnpEff v4.1.
