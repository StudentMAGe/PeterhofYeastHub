Description

These tracks feature annotations of genes on each contig or pseudochromosome.

For more information on assemblies please see the description page.

Methods

Initial sets of annotations (labeled 'MAKER2') were united using the MAKER2 pipeline v2.31.8 [Mark and Yandell et al., 2011].

To restore strandard gene names, genes found with this pipeline were divided into novel and known ones with an in-house python script available at github. This script employs a renaming table produced with proteinortho [Lechner et al., 2011] by clustering the whole sets of ORFs from the reference strain (S288C) and the strain analyzed. Tracks with renamed genes are labeled 'MAKER 2 renamed'.

Please note that standart yeast ORF names (e.g. YBR043C) are indexed and can be used for searching.
